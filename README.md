# EngagePulse ![Build Status](https://img.shields.io/badge/build-passing-brightgreen) ![Version](https://img.shields.io/badge/version-1.0.0-blue)

## Project Description
EngagePulse is a beginner-friendly web application designed to help small businesses automate their email marketing efforts. Users can easily create and send email campaigns, track their performance, and utilize pre-designed templates to enhance their outreach.

## Features
- 🌟 User-friendly interface for creating simple email campaigns
- 📊 Basic analytics dashboard to track open and click rates
- 🎨 Template library for quick email design

## Tech Stack
### Frontend
- **Node.js** 🟢

### Database
- **MongoDB** 🐱‍💻

## Installation
To set up the project locally, follow these steps:

- Clone the repository
bash
git clone https://github.com/arun-kumar-codes/engagepulse
- Navigate into the project directory
bash
cd engagepulse
- Install the required dependencies
bash
npm install
- Set up your MongoDB database (ensure MongoDB is running)
- Create a `.env` file and add your database connection string
- Start the application
bash
npm start
## Usage
Once the application is running, navigate to `http://localhost:3000` in your web browser. You can create email campaigns, view analytics, and access the template library from the dashboard.

## API Documentation
Currently, EngagePulse does not expose a public API. Future versions may include API endpoints for integration with other services.

## Testing
To run tests for the application, use the following command:
